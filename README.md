# 分镜式编辑器

## 项目简介
本项目无 ai,是离线纯网页
分镜式编辑器是一款功能强大的可视化编辑工具，专为创意设计和多媒体内容创作而打造。它提供了直观的界面和丰富的功能，让用户能够轻松创建和编辑分镜、图像、音频和视频内容。

## 核心功能

### 1. 分镜管理
- 新建分镜节点
- 清空所有分镜
- 定位到第一个节点
- 节点之间的连接管理

### 2. 图片编辑
- 图片裁剪功能
- 旋转、缩放和翻转调整
- 图层操作（上移、下移、置顶、置低）
- 图片调色功能
- 坐标精确调整

### 3. 绘画工具
- 无限画布模式
- 画笔工具（可调整颜色和大小）
- 橡皮擦工具
- 表情插入功能
- 绘画内容清空

### 4. 多媒体支持
- 音频系统（按钮音效）
- 音乐系统
- 视频系统

### 5. 项目管理
- JSON格式导入导出
- 代码合成和预览
- 复制代码功能

### 6. 用户界面
- 浮动工具栏
- 模态对话框
- 迷你地图导航
- 操作指南帮助

## 技术特点

- 基于HTML5 Canvas和SVG技术
- 纯前端实现，无需后端服务
- 模块化设计，代码结构清晰
- 响应式布局，适配不同屏幕尺寸
- 流畅的用户交互体验

## 使用指南

1. **启动项目**：直接在浏览器中打开 `share.html` 文件
2. **新建分镜**：点击"新建"按钮创建新的分镜节点
3. **编辑内容**：使用右侧控制面板调整图片属性
4. **添加绘画**：使用底部绘画工具栏进行手绘
5. **导出项目**：点击"导出"按钮保存为JSON文件
6. **合成代码**：点击"合成"按钮生成并预览代码




## 浏览器兼容性

- 推荐使用 Chrome、Firefox、Safari 等现代浏览器
- 支持 HTML5 Canvas 和 SVG 的浏览器均可运行



# Storyboard Editor

## Project Introduction
This project has no AI; it is an offline purely web-based application.

Storyboard Editor is a powerful visual editing tool designed for creative design and multimedia content creation. It provides an intuitive interface and rich features, allowing users to easily create and edit storyboards, images, audio, and video content.

## Core Features

### 1. Storyboard Management
- Create new storyboard nodes
- Clear all storyboards
- Locate to the first node
- Connection management between nodes

### 2. Image Editing
- Image cropping functionality
- Rotation, scaling, and flipping adjustments
- Layer operations (move up, move down, bring to top, send to bottom)
- Image color adjustment
- Precise coordinate adjustment

### 3. Drawing Tools
- Infinite canvas mode
- Brush tool (adjustable color and size)
- Eraser tool
- Emoji insertion functionality
- Clear drawing content

### 4. Multimedia Support
- Audio system (button sound effects)
- Music system
- Video system

### 5. Project Management
- JSON format import/export
- Code synthesis and preview
- Code copying functionality

### 6. User Interface
- Floating toolbar
- Modal dialogs
- Mini-map navigation
- Operation guide help

## Technical Features

- Based on HTML5 Canvas and SVG technologies
- Pure front-end implementation, no backend services required
- Modular design with clear code structure
- Responsive layout, adapting to different screen sizes
- Smooth user interaction experience

## Usage Guide

1. **Start the project**: Open `share.html` file directly in the browser
2. **Create new storyboard**: Click the "新建" button to create a new storyboard node
3. **Edit content**: Use the right-side control panel to adjust image properties
4. **Add drawing**: Use the bottom drawing toolbar for freehand drawing
5. **Export project**: Click the "导出" button to save as JSON file
6. **Synthesize code**: Click the "合成" button to generate and preview code



## Browser Compatibility

- Recommended to use modern browsers like Chrome, Firefox, Safari
- Compatible with any browser that supports HTML5 Canvas and SVG
